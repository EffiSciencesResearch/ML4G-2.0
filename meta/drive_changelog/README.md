# Drive Changelog

**Status:** currently unused. Kept around because we may revive it.

Originally a tool to monitor a Google Drive folder tree and report changes (file additions, edits, renames). It lives here so we don't lose the code while we decide whether to bring it back or replace it.

If you do want to run it:

```shell
make run-changelog
# or
uv run python -m meta.drive_changelog
```
